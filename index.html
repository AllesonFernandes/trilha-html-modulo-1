Estruturas semânticas em HTML

As estruturas semânticas trazem consigo uma ideia de sentido envolto de um contexto. Exemplo disso, vemos o header como uma unidade semântica que indica um cabeçalho, encabeçar uma página.

Construção de tabelas

A construção de tabelas facilita a identificação de dados dispostos em linhas e colunas. Usa-se tags para criar as quantidade de linhas como tr. Dentro da mesma encontramos outras tags como td e th. Todas elas devem estar envoltas das tags de abertura e fechamento table. Além disso, existem outras tags como thead, tbody e tfoot indicando semânticamente o que cada uma propõe.

Usando formulários

Os formulário são geralmente utilizados com a finalidade de obtenção de dados através de inputs. Os formatos desses inputs podem se adequar aos dados que são solicitados como texto, números, data e etc. Além disso, possuem os identificadores como as labels.
